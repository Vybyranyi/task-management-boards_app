# Task Management Frontend

Frontend додатку для управління завданнями.

## Технології

- React
- TypeScript
- Redux Toolkit

## Запуск локально

1. Встановіть залежності:
```bash
npm install
```

2. Створіть файл `.env`:
```
VITE_API_URL=http://localhost:3000/api
```

3. Запустіть проект:
```bash
npm run dev
```

## Функції

- Створення та завантаження дошок
- 3 колонки: To Do, In Progress, Done
- Drag and drop карток
- Додавання, редагування та видалення карток
- Поділитися дошкою через ID
